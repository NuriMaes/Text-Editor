# Python Text Editor

A simple and functional GUI text editor built with Python and Tkinter.

## Features
* Create and edit text files.
* Open existing `.txt` files.
* Smart Save logic (persistence of file paths).
* Clean and intuitive User Interface.

## Technologies Used
* **Python 3.x**
* **Tkinter** (Standard GUI Library)

## How to run
1. Clone the repository.
2. Run `python main.py`.
